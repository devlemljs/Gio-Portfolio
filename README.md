# Gio Sabucido Portfolio

A modern, responsive portfolio website built with React and TypeScript to showcase my work as an Amazon Virtual Assistant, Social Media Marketer, Graphic Designer, and Administrative Support Specialist.

## Overview

This project serves as my personal portfolio, highlighting my professional experience, skills, projects, and contact information. The application is built as a single-page application (SPA) with a focus on performance, accessibility, responsive design, and maintainable code.

## Tech Stack

- React
- TypeScript
- Vite
- HTML5
- CSS3

## Features

- Responsive layout for desktop, tablet, and mobile devices
- Smooth single-page navigation
- Professional portfolio sections
- Project showcase
- Skills overview
- Contact section
- SEO-friendly metadata
- Fast development with Vite
- Type-safe development using TypeScript

## Getting Started

### Prerequisites

- Node.js 18 or later
- npm

### Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio.git
```

Navigate to the project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

## Available Scripts

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

Run linting:

```bash
npm run lint
```

## Production Build

Generate an optimized production build:

```bash
npm run build
```

The compiled assets will be generated inside the `dist/` directory.
